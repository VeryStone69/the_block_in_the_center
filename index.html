7 ways to center align a block

text-align

css:

text-align: center; (for white div)

display: inline-block; (for blue div)

margin-auto

css:

margin: 10px auto; (for green div)

absolute

css:

position: relative; (for white div)

position: absolute; left: 50%; margin-left: -37.5px; (for orange div)

absolute 2

css:

position: relative; (for white div)

position: absolute; left: 50%; transform: translate(-50%,0); margin-left: 0; (if there is another margin) (for orange div)

flexbox

css:

display: flex; align-items: center; justify-content: center; (for white div)

grid

css:

display: grid; grid-template-areas: "a"; (for white div)

grid-area: a; align-self: center; justify-self: center; (for darkslategray div)
